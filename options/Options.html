SpreadsheetCloudAPI(link) uses an API key-based mechanism to identify applications. Either select "Use an existing API key" if you have an active SpreadsheetCloudAPI API key or select “Generate a new API key” to create a new SpreadsheetCloudAPI account for your e-mail and generate an API key within this account. In case of a new account, SpreadsheetCloudAPI will send you an e-mail with generated login and password for authorization within the service web interface.
